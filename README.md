# NoopyTV Cast Receiver

Custom Chromecast Receiver for NoopyTV IPTV application.

## Features

- HLS live stream support with TS segment format
- IPTV compatibility for various stream types
- Loading spinner while buffering

## Setup

1. Host this on GitHub Pages
2. Register on [Cast Developer Console](https://cast.google.com/publish)
3. Create a Custom Receiver app with this URL
4. Use the App ID in the iOS app

## Files

- `receiver.html` - The Chromecast receiver application
